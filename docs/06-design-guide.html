1 デザインコンセプト

デザインキーワード 必須
デザインの方向性を表すキーワード3つとその説明

上質感
落ち着いたベージュやブラウンを基調とした配色と広い余白によって、香水ブランドらしい上品で高級感のある印象を表現している。

和モダン
浅草という立地や日本の美意識に合わせて、自然や静けさを感じるビジュアルを取り入れ、伝統と現代性を融合したデザインになっている。

体験重視
香水づくりの流れや予約導線を分かりやすく整理し、ユーザーが安心して体験を予約できる構成を重視している。

4 UIコンポーネント

ボタン 必須
プライマリ・セカンダリ・デンジャーの3種類のボタンスタイル（実物のHTMLサンプル付き）

プライマリーボタン
予約や申込みなどの主要アクションに使用する。グラデーション背景と丸みのある形状により、視認性と高級感を両立している。

セカンダリーボタン
明るい背景と細いボーダーを用いたボタンで、補助的な操作や詳細確認に使用する。主張を抑えつつも視認性を確保している。

リンクボタン
ヒーローエリアなどで使用されるテキストベースのボタンで、軽い操作やページ遷移に使用する。シンプルで洗練された印象を与える。

HTMLサンプル
<a class="hero-cta">Reserve your session</a>
<a class="pricing-button secondary">Ask on site</a>
<a class="hero-link">Explore experiences</a>

CSSサンプル
.hero-cta {
  background: linear-gradient(135deg, #96756d 0%, #634b45 100%);

  border-radius: 999px;
  padding: 12px 24px;
}

.pricing-button.secondary {
  background: #fffdfb;

.hero-link {

フォーム要素 必須
テキスト入力・セレクト・テキストエリア・チェックボックス・ラジオボタン（実物のHTMLサンプル付き）

フォーム要素
現在のページには入力フォームは実装されていない。予約はボタンを通じて外部または別導線で行う構成となっている。

カード・テーブル・その他 必須
カードコンポーネント・テーブルスタイル・アラート等（実物のHTMLサンプル付き）

カード
明るい背景色と大きな角丸、柔らかいシャドウを用いて、情報を整理しながら上品な印象を与えている。

FAQアコーディオン
質問と回答を折りたたみ形式で表示し、情報量を整理しながらユーザーが必要な情報にアクセスしやすくしている。

情報カード
アクセス情報や体験内容はカード形式で表示され、視認性と理解しやすさを高めている。

5 レイアウトルール

レイアウト基本ルール 必須
グリッド・余白・角丸・シャドウ等の数値ルール

グリッド
1カラム、2カラム、3カラムのレイアウトをセクションごとに使い分け、画面幅に応じて柔軟に変化するレスポンシブ設計とする。

コンテナ幅
最大幅1200pxで中央配置し、読みやすさとバランスの取れたレイアウトを実現している。

余白
セクションごとに広い余白を取り、ゆったりとした高級感のあるデザインを構成している。

角丸
コンポーネントには大きめの角丸を使用し、柔らかく上品な印象を統一している。

シャドウ
薄く広がるシャドウを使用し、要素に奥行きを持たせながら軽やかなデザインにしている。

レスポンシブ
スマートフォン、タブレット、デスクトップに対応し、各デバイスで最適な表示を実現している。
