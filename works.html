Perfume Workshop Site（香水体験予約LPサイト））

作品概要 必須
どんなサービス・サイトか、ターゲットユーザーは誰か

オリジナル香水の魅力を伝え、ユーザーが安心して体験予約できることを目的としたLPサイトです。
ターゲットは、香水や体験型サービスに興味のある20代〜30代の男女を想定しました。
ブランドの高級感と世界観を表現するために、落ち着いた配色や余白を活かしたデザインを採用し、ユーザーが直感的に操作できるようシンプルで分かりやすい構成を意識しました。

使用技術 必須

フロントエンド：HTML / CSS / JavaScript（レスポンシブ対応）
レスポンシブ対応（スマートフォン・タブレット対応）
デザイン： figma

担当範囲 必須
企画・デザイン・フロントエンド・バックエンドなど

企画 / ワイヤーフレーム作成 / デザイン（Figma） / コーディング（HTML・CSS・JavaScript）

工夫した点・アピールポイント 必須

ユーザーが迷わず目的の情報にたどり着けるよう、シンプルで分かりやすいレイアウト設計を意識しました。
また、香水ブランドの世界観を表現するために、落ち着いた配色や余白を意識し、視覚的な統一感を重視しました。
CTA（予約ボタン）は目立つ位置に配置し、ユーザーがスムーズに行動できるよう工夫しています。
さらに、レスポンシブ対応を行い、スマートフォンでも快適に閲覧できるよう最適化しました。
